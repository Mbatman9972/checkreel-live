üîê Authentication

CheckReel API uses Bearer Token authentication with role-based access control.

Admin Dashboard Access

POST /auth/admin
Content-Type: application/json
{
  "password": "MM7268",
  "session_type": "admin_dashboard"
}

Response:

{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "token_type": "Bearer",
  "expires_in": 3600,
  "scope": "admin:dashboard emergency:controls tier:management",
  "user": {
    "role": "admin",
    "permissions": ["scan:unlimited", "tier:modify", "emergency:activate"]
  }
}

User Session Management

POST /auth/signin
{
  "email": "user@example.com",
  "language": "en|fr|ar",
  "ip_address": "192.168.1.100"
}

Headers Required

Authorization: Bearer {access_token}
Content-Type: application/json
Accept: application/json
X-API-Version: 1.0

        region='middle-east',
        country='uae'
    )

print(f"Compliance Score: {scan.compliance_score}")

üöÄ Quick Start

5-Minute Setup

  1. Get API Access: Sign up at CheckReel Dashboard
  2. Authenticate: Use your email to get session token
  3. Scan Content: Upload file with platform/region selection
  4. Get Results: Retrieve compliance score and recommendations
  5. Upgrade Tier: Access advanced features with Plus/Premium

Complete Example

# 1. Authenticate
curl -X POST https://api.checkreel.com/v1/auth/signin \
  -H "Content-Type: application/json" \
  -d '{"email": "user@example.com"}'

# 2. Scan content
curl -X POST https://api.checkreel.com/v1/scan \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "file=@video.mp4" \
  -F "platform=tiktok" \
  -F "region=middle-east" \
  -F "country=uae"

# 3. Get results
curl -X GET https://api.checkreel.com/v1/scan/scn_9876543210 \
  -H "Authorization: Bearer YOUR_TOKEN"
