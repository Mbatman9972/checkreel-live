ğŸ” Content Scanning

Initiate Content Scan

Analyze content for platform compliance across multiple regions and platforms.

POST /scan
Authorization: Bearer {token}
Content-Type: multipart/form-data

file: [binary content]
platform: ["tiktok", "youtube", "instagram", "facebook", "twitter", "snapchat"]
region: "middle-east|global|europe|mena"
country: "uae|saudi|usa|uk|france"
priority: "normal|high|critical"

Supported File Types

File Type Formats Size Limit Duration Limit
Video MP4, MOV, AVI 100MB 10 minutes
Audio MP3, WAV 50MB 15 minutes
Image JPG, PNG 25MB N/A
