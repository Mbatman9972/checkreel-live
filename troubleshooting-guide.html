?? Upload & Scanning Issues

"Please select region and platform" Error

CRITICAL REQUIREMENT: You must select both region and platform before uploading any content.

Step-by-Step Solution:

  1. Select Region: Middle East, Global, Europe, or MENA
  2. Select Country: Choose your specific country
  3. Select Platform: Choose at least one platform (TikTok, Instagram, etc.)
  4. Upload: Now you can upload your content

File Type Errors

Supported File Formats:

  • Video: MP4, MOV, AVI (max 100MB, 10 minutes)
  • Audio: MP3, WAV (max 50MB, 15 minutes)
  • Image: JPG, PNG (max 25MB)

Solution: Convert your file to a supported format or reduce file size.

Scanning Stuck or Takes Too Long

Possible Causes:

  • File size too large
  • Server overload
  • Network connectivity issues

Solutions:

  1. Refresh the page and try again
  2. Reduce file size (compress video/audio)
  3. Check your internet connection
  4. Try again during off-peak hours

?? Admin Dashboard Problems

Cannot Access Admin Dashboard

Admin Button Not Visible

To reveal admin button:

  1. Click the CheckReel logo (?? CheckReel Control Room) exactly 5 times quickly
  2. Admin button (?? Admin) will appear in the header
  3. Click the Admin button

Wrong Password Error

Admin Password: MM7286CR

IMPORTANT: Password is case-sensitive and must be entered exactly as shown.

Troubleshooting Steps:

  1. Verify password: MM7286CR (not MM7268)
  2. Clear browser cache and cookies
  3. Try in incognito/private browsing mode
  4. Refresh page after clearing cache

Admin Features Not Working

Supabase Connection Issues:

If KPIs show "Loading..." or "Error", the Supabase backend connection may be down.

Check Supabase Status:

  1. Open browser developer tools (F12)
  2. Check Console tab for errors
  3. Look for Supabase API errors (401, 403, 500)
  4. Contact support if errors persist

?? Tier & Billing Issues

Scan Limit Problems

Tier Limits:

  • Free: 3 scans per month
  • Plus ($4.99): 20 scans per month
  • Premium ($9.99): 40 scans per month

When limit exceeded:

  1. Upgrade to higher tier for immediate access
  2. Wait for monthly reset (1st of each month)
  3. Contact support for emergency increase

Payment & Upgrade Issues

Common Payment Failure Reasons:

  • Insufficient funds
  • Card expired or invalid
  • Bank declined transaction
  • Stripe processing issues

Solutions:

  1. Verify card details are correct
  2. Contact your bank to authorize payment
  3. Try a different payment method
  4. Contact support if issues persist

?? Language & Display Issues

Wrong Language Displaying

Available Languages:

  • 🇺🇸 English - Complete features
  • 🇸🇦 العربية - RTL support + Arabic numerals

To Change Language:

  1. Find language selector in top-right header
  2. Select your preferred language
  3. Page updates immediately
  4. Your preference is saved automatically

RTL (Right-to-Left) Display Issues

CheckReel fully supports Arabic RTL display with proper text direction and Arabic numerals.

If RTL display appears broken:

  1. Clear browser cache completely
  2. Refresh the page
  3. Try a different browser
  4. Disable browser extensions temporarily

?? Emergency Recovery

?? When to Use Emergency Recovery

Use ONLY when the admin dashboard is completely broken and normal troubleshooting has failed.

Browser-Level Recovery

# Clear All Site Data
1. Open Chrome/Firefox Settings
2. Go to Privacy & Security
3. Clear Browsing Data
4. Select "All Time" and check all boxes
5. Clear Data and restart browser

Emergency Mode Access

If you have admin access, the Emergency Mode in admin dashboard provides system override capabilities:

  • Force stop all scanning operations
  • Kill active API connections
  • Database maintenance mode
  • System recovery tools
